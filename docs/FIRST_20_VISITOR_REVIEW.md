## Decision Gate

After 20 sessions:

- If pass rate on "understood business in 5 sec" < 70% → homepage hero requires rewrite before any other work
- If pass rate on "trusted founders" < 70% → founders section requires redesign or expanded copy
- If pass rate on "would submit inquiry" < 50% → inquiry path or friction point requires immediate attention
- If mobile checks have 2+ failures → mobile layout requires fix before any outreach

No feature development begins until this gate is reviewed.
